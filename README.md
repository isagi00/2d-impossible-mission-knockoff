# 2d-impossible-mission-knockoff
project made for the Sapienza's university first year java course, Metodologie di Programmazione. 
It is knockoff of the C64 game "Impossible Mission", revamped with new gameplay techs and graphics.
This projects follows an MVC architecture, with design patterns such as Observer and Singleton.

first ever project, made with prayers, caffeine and nicotine.
